Read this first. Per-start ER is one of the noisiest targets in baseball (BABIP/sequencing/luck dominate). "80% accuracy" is not attainable as exact ER. The honest, concrete metric is the ±1 ER band hit-rate (realistic ceiling ~60–65%); true 80%+ only appears in the low-information ±2 band. Every number below is shown next to a season-rate baseline so the model's marginal value is unambiguous.
